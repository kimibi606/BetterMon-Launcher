    Java 실행 파일

    유효하지 않은 선택

    게임 실행 전에 Java 실행 파일의 유효성을 검사합니다. 64비트 Java 21이 필요합니다.
    Java 실행 파일의 경로는 보통 bin/javaw.exe로 끝나야 합니다.

    JVM 인수 설정

    실행 시 JVM에 전달할 추가 인수를 설정합니다. -Xms와 -Xmx는 포함되지 않아야 합니다.

    Java 21 (HotSpot VM) 기준으로 동작하는 JVM 인수를 사용하는 것을 권장합니다.
